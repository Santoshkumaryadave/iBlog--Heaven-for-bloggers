There is no need to download or use any complicated software or technical know-how for blog creation on this. To create their blog on it, users only have to create their account on Google. To signup on this, the user has to put a different name which will be the name of his blog. The same name is also used as a domain name. [2] BlogSpot doesn't have features like some other blogging services, but its simple tools and other features make blogging super easy. But this does not mean that Blogspot does not have modern facilities. Most advanced users can make a number of changes to their blog, including designing their own template and even using Google's website analytics program . several thousand tools on google (gadgets ) that can be attached to blogs. With their help, bloggers can create multi-user blogs, with the help of which many bloggers can contribute on the same blog.
